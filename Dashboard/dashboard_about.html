Sistema de controle Touchless

O projeto utiliza 5 sensores de distância em formato de cruz para capturar movimentos do usuário e gerar os comandos para serem interpretados. Para demonstração nesse projeto, configuramos um mouse, embora existam outras aplicações como por exemplo um controle gestual em serviços de auto-atendimento.

Mouse Touchless

Para movimentar o mouse em uma direção deve-se posicionar sua mão próximo ao sensor na direção desejada, e usar o sensor central para selecionar.
